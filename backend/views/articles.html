GET /api/articles

Retrieve all articles from the blog

API Documentation

{{infoJson}}

How to Use

This endpoint returns a list of all articles. Simply make a GET request.

Using curl:

curl http://localhost:3001/api/articles

Using fetch (JavaScript):

fetch('http://localhost:3001/api/articles')
  .then(res => res.json())
  .then(data => console.log(data));
