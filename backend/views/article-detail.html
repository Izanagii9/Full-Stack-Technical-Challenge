GET /api/articles/:id

Retrieve a specific article by its ID

API Documentation

{{infoJson}}

How to Use

Replace :id with the article ID you want to retrieve.

Using curl:

curl http://localhost:3001/api/articles/{{id}}

Using fetch (JavaScript):

fetch('http://localhost:3001/api/articles/{{id}}')
  .then(res => res.json())
  .then(data => console.log(data));
